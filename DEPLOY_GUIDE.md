## 🌐 デプロイ手順（Cloudflare Pages）

### 前提条件
- Cloudflareアカウント（既にviola-pure.xyzがデプロイされている）
- GitHubリポジトリへのアクセス権
- DATABASE_URL環境変数が設定済み

### ステップ1: Cloudflareダッシュボードにアクセス

1. https://dash.cloudflare.com/ にログイン
2. 左サイドバーから **Workers & Pages** をクリック
3. **viola-pure** プロジェクトを選択

### ステップ2: 自動デプロイを確認

GitHubに`main`ブランチをプッシュすると、Cloudflare Pagesが自動的にビルド・デプロイします。

1. **Deployments** タブを開く
2. 最新のデプロイが「**In progress**」または「**Success**」になっていることを確認
3. ビルドログを確認（エラーがある場合はログを確認）

### ステップ3: デプロイ完了を確認

デプロイが完了したら（通常3〜5分）、以下のURLにアクセスして確認：

```
https://www.viola-pure.xyz/admin/export
```

新しいタブが表示されるはずです：
- 🌲 MLMボーナス
- 📱 携帯契約
- ✈️ 旅行サブスク
- 💰 振込データ

## 🧪 機能テスト

### 1. MLMボーナスCSVエクスポート

1. 管理画面にログイン: https://www.viola-pure.xyz/admin/login
2. 左サイドバー → **📥 CSV エクスポート**
3. **🌲 MLMボーナス** タブをクリック
4. 対象月を選択（例: 2026-02）
5. **⬇️ MLMボーナス CSV をダウンロード** ボタンをクリック
6. ダウンロードされたCSVファイルを確認

### 2. 携帯契約CSVエクスポート

1. **📱 携帯契約** タブをクリック
2. ステータスを選択（任意）
3. **⬇️ 携帯契約 CSV をダウンロード** ボタンをクリック

### 3. 旅行サブスクCSVエクスポート

1. **✈️ 旅行サブスク** タブをクリック
2. ステータスを選択（任意）
3. **⬇️ 旅行サブスク CSV をダウンロード** ボタンをクリック

### 4. ウェブフリコム振込データ出力

1. **💰 振込データ** タブをクリック
2. 対象月を選択（例: 2026-02）

4. ダウンロードされたテキストファイルを確認
   - ファイル形式: `202602.txt`（月の数字のみ）
   - 各行120文字の固定長フォーマット
   - エンコーディング: Shift_JIS

## 📋 ウェブフリコム振込データ仕様

### ファイルフォーマット

```
行番号 | 内容           | 長さ
-------|---------------|------
1      | ヘッダー行     | 120文字
2〜N   | データ行       | 120文字
N+1    | トレーラ行     | 120文字
N+2    | エンド行       | 120文字
```

### ヘッダー行（1行目）
```
委託者コード (14桁) + 空白 (90桁) + 振込指定日 (8桁) + 空白 (8桁)
```

### データ行（2〜N行目）
```
銀行コード (5桁)
+ 空白 (15桁)
+ 支店コード (3桁)
+ 空白 (19桁)
+ 会員コード (8桁)
+ 受取人名 (カナ30桁)
+ 空白 (6桁)
+ 金額 (10桁)
+ 新規コード (1桁)
+ ゼロ (12桁)
+ 会員コード下7桁 (7桁)
+ ダミーコード (1桁)
+ 新規コード (1桁)
+ ゼロ+空白 (16桁)
```

### トレーラ行（最終行-1）
```
"8" + 件数合計 (6桁) + 金額合計 (12桁) + 空白 (101桁)
```

### エンド行（最終行）
```
"9" + 空白 (119桁)
```

## ⚠️ トラブルシューティング

### ビルドエラーが発生した場合

1. Cloudflareダッシュボード → **Deployments** → エラーのあるデプロイをクリック
2. ビルドログを確認
3. よくあるエラー:
   - **DATABASE_URL環境変数が未設定**: Settings → Environment variables で設定
   - **依存関係エラー**: `package.json`の依存関係を確認
   - **TypeScriptエラー**: ローカルで`npm run build`を実行して確認

### データが表示されない場合

1. **ボーナス計算が確定されているか確認**
   - `/admin/bonus-run` ページで対象月のボーナス計算が「確定済み」になっているか
   - 「下書き」の場合は確定処理を実行

2. **口座情報が登録されているか確認**
   - `/admin/mlm-members` ページで会員の口座情報を確認
   - 口座番号、口座名義が登録されていない場合はウェブフリコムデータに含まれません

3. **データベース接続を確認**
   - Settings → Environment variables → `DATABASE_URL`が正しく設定されているか

### ダウンロードしたファイルが文字化けする場合

- **ウェブフリコムデータ**: Shift_JIS対応エディタで開く（メモ帳、サクラエディタなど）
- **CSVファイル**: Excel、Googleスプレッドシートで開く（BOM付きUTF-8で出力されています）
